# 《统计学习方法》- 李航

## 📖 基本信息
- **作者**: 李航（前华为诺亚方舟实验室主任）
- **出版年份**: 第一版 2012，第二版 2019
- **核心主题**: 统计机器学习方法的严格数学推导——从感知机到 CRF 的 11 种经典算法
- **目标读者**: 希望扎实理解 ML 数学原理的工程师；研究方向的学习者
- **阅读难度**: expert

## 🎯 核心价值
> 每个统计学习方法 = 模型 + 策略 + 算法——这个三元素框架让你理解任何 ML 方法的本质，而不是只会调包。

## 📚 知识框架

### 主要章节/模块

### 统计学习三要素

```
模型（Model）：假设空间 F，即可能的函数集合
策略（Strategy）：损失函数 L + 正则化，定义"什么是好"
算法（Algorithm）：求解最优化问题的具体方法
```

### 1. 感知机（Perceptron）
- 二类线性分类器；误分类驱动更新
- **学习算法**：随机梯度下降；对误分类点更新 w 和 b
- **对偶形式**：用 Gram 矩阵（内积）表示，为 SVM 对偶铺垫
- **收敛性**：线性可分时必然收敛

### 2. K 近邻法（KNN）
- 非参数方法，无显式训练；"懒惰学习"
- **距离度量**：Minkowski 距离（L1/L2 是特例）
- **k 的选择**：k 小则过拟合；k 大则欠拟合
- **KD 树**：加速 KNN 查询的空间数据结构

### 3. 朴素贝叶斯（Naive Bayes）
- **条件独立假设**：`P(X|Y) = ∏P(Xⱼ|Y)`（极大简化计算）
- **最大后验估计**：`arg max P(Y|X) = arg max P(Y)·P(X|Y)`
- **拉普拉斯平滑**：处理零概率问题（分子 +1，分母 +K）
- 适合：文本分类、垃圾邮件过滤

### 4. 决策树
- **特征选择准则**：信息增益（ID3）/ 信息增益比（C4.5）/ 基尼系数（CART）
- **信息增益 = H(D) - H(D|A)**：特征 A 对数据集 D 的不确定性减少量
- **CART**：二叉树，回归用均方误差，分类用基尼系数
- **剪枝**：极小化决策树整体损失函数 C_α(T) = C(T) + α|T|

### 5. 逻辑回归与最大熵模型
- **逻辑回归**：对数几率模型；等价于条件概率的最大熵模型
- **最大熵原理**：在满足约束条件的所有模型中，熵最大的模型是最好的模型
- **最大熵模型**：特征函数约束下的概率分布；用 IIS 或 BFGS 求解

### 6. 支持向量机（SVM）— 重点章节

#### 线性可分 SVM
- **间隔最大化**：找使支持向量到超平面距离最大的 w, b
- **原始问题**：`min 1/2||w||²  s.t. yᵢ(w·xᵢ+b) ≥ 1`
- **对偶问题**：通过拉格朗日对偶转化，只用内积计算

#### 软间隔 SVM
- 引入松弛变量 ξ 允许误分类
- 正则化参数 C：C 大则容忍误分类少（复杂边界）；C 小则间隔大（简单边界）

#### 核技巧
- **核函数 K(x,z) = φ(x)·φ(z)**：不显式计算高维映射，只计算内积
- **常用核**：
  - 多项式核：`K(x,z) = (x·z+1)^p`
  - 高斯核（RBF）：`K(x,z) = exp(-||x-z||²/2σ²)`
  - 字符串核：用于文本序列

#### SMO 算法
- 每次选择两个变量优化，其余固定
- 解析求解二变量子问题

### 7. Boosting（提升方法）

#### AdaBoost
```
思想：用加权多数表决组合弱分类器
流程：
1. 初始化样本权重（均匀分布）
2. 训练弱分类器，计算误差率 eₘ
3. 计算分类器权重 αₘ = 1/2 ln((1-eₘ)/eₘ)
4. 更新样本权重（误分类样本增加权重）
5. 组合：F(x) = sign(Σαₘ·Gₘ(x))
```

#### 提升树与 GBDT
- **提升树**：以决策树为基学习器
- **GBDT（梯度提升决策树）**：用梯度下降的思路拟合残差
- **XGBoost**：GBDT 的工程优化版本，加入正则化和并行化

### 8. EM 算法
- **适用场景**：存在隐变量的概率模型参数估计
- **E 步**：计算完全数据对数似然关于隐变量的期望 Q 函数
- **M 步**：极大化 Q 函数，更新参数
- **典型应用**：高斯混合模型（GMM）、HMM 训练

### 9. 隐马尔可夫模型（HMM）
- **三个问题**：
  1. 概率计算（前向/后向算法）
  2. 学习（Baum-Welch = EM）
  3. 预测（Viterbi 算法，动态规划）
- **应用**：语音识别、词性标注、生物序列分析

### 10. 条件随机场（CRF）
- **判别式模型**：直接建模 P(Y|X)，比 HMM 更灵活
- **线性链 CRF**：序列标注任务
- **特征函数**：可以是任意关于观测序列和标注的函数
- **应用**：命名实体识别（NER）、中文分词

## 🔑 关键概念

### 生成模型 vs 判别模型
**定义**: 
- 生成模型：学习 P(X,Y) 联合分布，可生成样本（朴素贝叶斯、HMM）
- 判别模型：直接学习 P(Y|X) 条件分布（逻辑回归、SVM、CRF）

**重要性**: 指导模型选择和理解模型能力边界

**应用场景**:
1. 数据少 + 需要先验知识 → 生成模型
2. 数据多 + 关注分类性能 → 判别模型
3. LLM 既是生成模型又可用于判别

---

### 结构风险最小化
**定义**: 总风险 = 经验风险（训练误差）+ 正则化项（模型复杂度惩罚）

**重要性**: 正则化的理论依据；L1/L2 正则化的数学来源

**应用场景**:
1. L1 正则化（Lasso）→ 产生稀疏解，特征选择
2. L2 正则化（Ridge）→ 权重均匀收缩，防止过拟合

## 💡 实战应用场景

### 场景 1: 理解 LLM 的预训练目标
**背景**: 需要向客户解释 GPT 的训练原理

**解决方案**: 用最大似然估计解释语言模型训练

**实施步骤**:
1. 语言模型 = 对文本序列概率分布的建模
2. 预训练 = 极大化训练语料的对数似然
3. 自回归 = P(x₁,...,xₙ) = ∏P(xᵢ|x₁,...,xᵢ₋₁)
4. 这等价于最小化交叉熵损失

**预期效果**: 用严格的数学语言讲清楚 LLM 的本质

---

### 场景 2: NER 标注系统选型
**背景**: 需要从非结构化文本中提取实体（人名、地名、产品名）

**解决方案**: CRF 或 BERT+CRF 用于序列标注

**实施步骤**:
1. 定义标注格式（BIO/BIOES）
2. 少量数据：CRF + 人工特征工程
3. 足量数据：BERT + CRF 联合训练

**预期效果**: 准确率 F1 90%+（领域数据充足时）
