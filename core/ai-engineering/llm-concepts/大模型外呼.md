# 大模型外呼 (LLM-Powered Voice Agent)

## 核心定义

**大模型外呼**：基于大语言模型(LLM)驱动的语音对话智能体，能真正理解用户意图、进行自然多轮对话、动态生成回复，区别于传统规则树或简单NLU驱动的外呼系统。

**核心理念**：用LLM作为"大脑"，ASR/TTS作为"耳朵/嘴巴"，实现接近真人的电话对话体验。

## 🎯 一句话总结

> **大模型外呼 = LLM全权掌控对话 + 流式语音交互 + 实时打断 + 动态生成 = 真人级电话交互体验**

## 📊 传统AI外呼 vs 大模型外呼

| 维度 | 传统AI外呼 | 大模型外呼 |
|------|-----------|-----------|
| 对话理解 | 关键词/意图穷举 | 真正语义理解 |
| 回复生成 | 预设话术库 | 动态生成 |
| 多轮能力 | 有限状态机 | 无限上下文 |
| 打断处理 | 生硬/无法打断 | 优雅打断+响应 |
| 异常处理 | 预设分支 | 未知情况也能应对 |
| 学习方式 | 人工补充话术 | LLM自动泛化 |
| 情感表达 | 固定语调 | 动态情感调整 |
| 成本/通 | 低 | 中高 (但转化率高) |

## 💼 实战案例

### 案例: 金融产品营销外呼

**背景**: 银行推广信用卡分期业务，需要筛选高意向客户

**传统AI外呼问题**:
- 客户说"我考虑一下" → 无法应对，只能结束
- 客户问具体利率 → 规则无法匹配
- 客户抱怨 → 直接挂机，无安抚

**大模型外呼方案**:
```
系统提示词:
你是一位专业的银行客户经理，正在电话营销信用卡分期业务。
- 语气专业、友好、有耐心
- 根据客户回应动态调整话术
- 遇到投诉先安抚，再引导
- 遇到复杂问题记录并转人工
- 每次回复不超过3句话
```

**效果对比**:

| 指标 | 传统AI | 大模型外呼 |
|------|--------|-----------|
| 平均通话时长 | 45秒 | 180秒 |
| 意向转化率 | 1.2% | 4.8% |
| 客户满意度 | 3.1/5 | 4.2/5 |
| 投诉率 | 8% | 2% |

### 案例: 课程咨询邀约

**背景**: 教育机构邀约潜在客户参加公开课

**大模型外呼核心话术策略**:
1. 开场: 提及客户姓名，建立亲近感
2. 需求挖掘: "您目前在学习什么？"
3. 痛点共鸣: "很多学员之前也有类似困惑"
4. 方案介绍: "我们的公开课专门针对..."
5. 异议处理: 动态生成个性化回答
6. 邀约成交: "您周六还是周日方便？"

## ⚠️ 核心挑战与解决方案

| 挑战 | 描述 | 解决方案 |
|------|------|----------|
| 延迟 | LLM生成慢，用户等待尴尬 | 预测性回复 + 快速TTS |
| 幻觉 | LLM胡说八道 | RAG锁定业务知识 + 拒答 |
| 成本 | GPT-4o成本高 | 小模型+领域微调 |
| 打断 | 打断后混乱 | VAD优先 + 对话状态重置 |
| 录音质检 | 无法事后规则质检 | 全量录音+实时分析 |
| 合规 | AI无法承诺/过量拨打 | 业务边界限制 + 人工兜底 |
