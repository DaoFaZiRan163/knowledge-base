# 外呼行业概述

## 核心定义

**外呼 (Outbound Call)**：由企业主动发起，向客户拨打电话的业务模式，区别于客户打进来的客服热线 (Inbound)。

**核心理念**：规模化触达客户，通过电话完成营销、通知、催收、调研等任务，核心指标是接通率、转化率和单人效率。

## 🎯 一句话总结

> **外呼 = 主动电话触达，核心挑战是"打给谁、说什么、怎么改"，决定效率的三要素是人、话术、数据**

## 📊 行业分类

| 类型 | 目标 | 特点 | 难度 |
|------|------|------|------|
| 营销外呼 | 推广产品/服务，产生购买 | 需求挖掘 + 产品推荐 | ⭐⭐⭐⭐ |
| 催收外呼 | 提醒/督促还款 | 合规要求高，语速快 | ⭐⭐⭐⭐⭐ |
| 通知触达 | 快递/预约/活动通知 | 话术固定，注重触达 | ⭐⭐ |
| 客服外呼 | 满意度回访，问题跟进 | 温和，关系维护 | ⭐⭐⭐ |
| 调研外呼 | 市场调研，数据收集 | 问卷结构化 | ⭐⭐ |

## 🏢 行业规模与现状

### 中国外呼市场

| 指标 | 数据 |
|------|------|
| 呼叫中心坐席数 | ~1000万+ |
| 行业市场规模 | ~3000亿元/年 |
| AI渗透率 | ~15-20% (快速提升) |
| 主要行业 | 金融(25%)、电商(20%)、运营商(18%)、政务(15%) |

### 传统外呼的痛点

| 痛点 | 描述 | 影响 |
|------|------|------|
| 成本高 | 每人每天100-150通，人力成本占80%+ | 盈利困难 |
| 效率低 | 大量无效号码、空号、拒接 | 浪费60%+人力 |
| 体验差 | 机械话术，标准化难 | 客户抵触 |
| 管理难 | 人员流动大，培训成本高 | 质量不稳定 |
| 合规难 | 骚扰电话、隐私保护 | 监管风险 |

### 政策监管

| 法规 | 要点 |
|------|------|
| 《个人信息保护法》 | 禁止未经同意的外呼 |
| 《通信短信息服务管理规定》 | 营销需明确同意 |
| 《互联网广告管理办法》 | 禁止AI伪装人工 |
| 96110防骚扰 | 标记和拦截高频外呼 |

## 💼 业务流程

### 标准外呼流程

```
1. 数据清洗 → 剔除空号/停机 → 有效号码列表
2. 线路预检 → 判断运营商/地区 → 优选线路
3. 自动拨号 → 系统批量外呼 → 等待接听
4. 接通判断 → 无人接/占线/拒接 → 标记重呼
5. 转人工/AI → 接通后转接 → 开始沟通
6. 意向标记 → 记录结果(A/B/C/D) → 更新CRM
7. 跟进策略 → 按意向分类处理 → 定时复触
```

### 核心指标体系

| 指标 | 定义 | 行业均值 | 优秀值 |
|------|------|----------|--------|
| 接通率 | 拨通数/拨打数 | 30-40% | >50% |
| 转化率 | 目标达成/接通数 | 1-3% | >5% |
| 有效通话率 | 通话>30秒/接通数 | 60-70% | >80% |
| 每人产能 | 有效通话/坐席/天 | 80-100通 | >150通 |
| 挂机率 | 3秒内挂断/接通 | 40-50% | <30% |
